# T-Shirt ⠞⠤⠎⠓⠊⠗⠞

![ThreeJS](https://img.shields.io/badge/Three.js-000000.svg?style=for-the-badge&logo=threedotjs&logoColor=white)
![ViteJS](https://img.shields.io/badge/Vite-9135FF.svg?style=for-the-badge&logo=Vite&logoColor=white)
![Rust](https://img.shields.io/badge/Rust-000000.svg?style=for-the-badge&logo=Rust&logoColor=white)

3D T-shirt design using ThreeJS and Rust running on the server. This project allows you to customize your 3D t-shirts and add your own textures.

This project came from a tutorial on the YouTube channel [LibreriasJS](https://www.youtube.com/@libreriasjs).

## Useful Links
