Design de Camisetas 3D usando ThreeJS e Rust rodando no servidor. Este projeto permite que você personalize suas camisetas 3D e adicione suas próprias texturas.

Este projeto foi inspirado em um tutorial do canal [LibreriasJS](https://www.youtube.com/@libreriasjs) no YouTube.
