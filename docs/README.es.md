Diseño de Camisetas 3D con ThreeJS y Rust, ejecutándose en el servidor. Este proyecto te permite personalizar tus camisetas 3D y añadir tus propias texturas.

Este proyecto se basa en un tutorial del canal de YouTube [LibreriasJS](https://www.youtube.com/@libreriasjs).
